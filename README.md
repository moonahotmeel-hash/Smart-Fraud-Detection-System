# Fraud Detection System using CNN-RNN-LSTM

A Hybrid Deep Learning System for Financial Fraud Detection using:

* CNN (Convolutional Neural Network)
* RNN (Recurrent Neural Network)
* LSTM (Long Short-Term Memory)

## Project Features

* Financial Fraud Detection
* Feature Engineering
* Data Balancing using SMOTE
* CNN-RNN-LSTM Hybrid Model
* Confusion Matrix
* ROC Curve
* Accuracy & Loss Curves
* Automated Model Training
* Gradio Web Application
* Deployment using GitHub and Render

## Dataset

Synthetic Financial Dataset for Fraud Detection.

## Technologies
